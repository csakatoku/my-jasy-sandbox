My Jasy Sandbox
================

JasyとCoreの使い方を練習するための個人的なレポジトリです。

- https://github.com/wpbasti/jasy
- https://github.com/wpbasti/core

現在のところ、jasy==0.4.6でしか動きません。

    $ pip install https://github.com/wpbasti/jasy/zipball/0.4.6

各プロジェクトのビルドの仕方は、

    $ cd helloworld
    $ jasy jasyscript.py build
